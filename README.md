**Netflix Recommendation System using TF-IDF and Cosine Similarity**

**Problem Statement**
With thousands of titles available on Netflix, users often find it difficult to discover relevant content.
This project aims to build a recommendation system that suggests similar movies and TV shows based on textual metadata.

**Objectives**
Analyze Netflix dataset
Clean and preprocess data
Convert textual content into numerical vectors using TF-IDF
Compute similarity between titles using cosine similarity
Recommend top similar movies/TV shows
Create interactive visualizations and dashboards

**Dataset**
Netflix Movies and TV Shows Dataset from Kaggle:
